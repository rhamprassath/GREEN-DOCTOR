# Green Doctor Backend

This is the backend for the Green Doctor application, an AI-powered plant disease detection system.

## 🚀 Deployment on Hugging Face Spaces

This repository is configured to run automatically on Hugging Face Spaces using Docker.

- **SDK**: Docker
- **Port**: 10000 (configured in `Dockerfile` and metadata above)
- **RAM Recommended**: 16GB (Free "CPU-basic" tier on Hugging Face)

## 🛠 Features

- **FastAPI**: High-performance backend.
- **AI-Powered**: Uses deep learning models for accurate leaf disease detection.
- **Dockerized**: Easy to deploy anywhere.
